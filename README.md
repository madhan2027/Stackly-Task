# Stackly Task - Personal Webpage

## 📌 Project Overview
This project is a simple personal webpage created using HTML and CSS as part of the Stackly training task.

## 🚀 Features
- Header with welcome message
- Navigation menu
- Home section
- About Me section
- Educational Qualification table
- Skills list
- Contact Form
- Footer

## 🛠️ Technologies Used
- HTML5
- CSS3

## 📂 Project Structure
```
Stackly-Task/
│── index.html
│── style.css
└── README.md
```

## ▶️ How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/madhan2027/Stackly-Task.git
   ```

2. Open the project folder.

3. Double-click `index.html` or open it in any web browser.

The webpage includes:
- Navigation Bar
- Home Section
